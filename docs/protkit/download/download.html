protkit.download.download

Implements the Download class to download biological data from the internet.

Currently, downloading of the following data is supported:

  • PDB files from the RCSB and Sabdab
  • CIF files from the RCSB
  • Binary CIF files from the RCSB
  • FASTA files from the RCSB and Uniprot

For more information about the various data sources, see the following URLs:

  • RCSB: https://www.rcsb.org/
  • Uniprot: https://www.uniprot.org/
  • Sabdab: https://opig.stats.ox.ac.uk/webapps/sabdab-sabpred/sabdab/

A list of download services provided by RCSB is available at:

  • https://www.rcsb.org/docs/programmatic-access/file-download-services

 41    @staticmethod
 42    def download_file(url: str, file_path: str) -> None:
 43        """
 44        Downloads a single file from the internet.
 45
 46        Uses the `requests` library to download the file.
 47
 48        Args:
 49            url (str): The URL of the file.
 50            file_path (str): The path to the file.
 51
 52        Returns:
 53            None
 54
 55        Raises:
 56            Exception: If the file could not be downloaded or saved.
 57        """
 58        try:
 59            response = requests.get(url)
 60            if response.status_code == 200:
 61                # Create the directory if it does not exist.
 62                directory = os.path.dirname(file_path)
 63                if directory != "":
 64                    if not os.path.exists(directory):
 65                        os.makedirs(directory)
 66
 67                # Write the file to disk.
 68                with open(file_path, "wb") as file:
 69                    file.write(response.content)
 70        except Exception as e:
 71            raise e

 73    @staticmethod
 74    def _download_file(url: str, file_path: str) -> None:
 75        """
 76        Downloads a single file from the internet.
 77
 78        Uses the `urllib` library to download the file.
 79
 80        Args:
 81            url (str): The URL of the file.
 82            file_path (str): The path to the file.
 83
 84        Returns:
 85            None
 86
 87        Raises:
 88            Exception: If the file could not be downloaded or saved.
 89        """
 90        import urllib.request
 91        try:
 92            with urllib.request.urlopen(url) as response:
 93                content = response.read()
 94
 95                with open(file_path, 'wb') as file:
 96                    file.write(content)
 97        except Exception as e:
 98            raise e

100    @staticmethod
101    def parallel_download(
102            urls: List[str],
103            file_paths: List[str],
104            n_jobs: int = -1) -> None:
105        """
106        Downloads multiple files from the internet in parallel.
107
108        Args:
109            urls (List[str]): The URLs of the files.
110            file_paths (List[str]): The paths to the files.
111            n_jobs (int): The number of jobs to run in parallel.
112                If -1, the number of jobs is set to the number of CPU cores.
113
114        Returns:
115            None
116
117        Raises:
118            Exception: If the files could not be downloaded or saved.
119        """
120        Parallel(n_jobs=n_jobs)(delayed(Download.download_file)(url, file_path)
121                                for url, file_path in zip(urls, file_paths))
